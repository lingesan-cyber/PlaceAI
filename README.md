# PLACE AI

## Overview

College Placement Management System

## Features

- Authentication
- Student Management
- Company Management
- Placement Tracking
- Training Management
- Dashboard Analytics
- Batch Year Management
- Spreadsheet Importer
- Export Features

## Tech Stack

Frontend:

- React
- TypeScript
- Tailwind CSS
- React Query

Backend:

- Node.js
- Express.js

Database:

- MongoDB

## Installation

Frontend:
npm install
npm run dev

Backend:
npm install
npm run dev

## Environment Variables

MONGO_URI=
JWT_SECRET=
JWT_EXPIRE=
